# 6.1 APIs

## Overview

In todayâ€™s lesson, students will be introduced to JSON traversal and the fundamentals of making API requests with the [Requests library](http://docs.python-requests.org/en/master/), using the [OMDb](https://www.omdbapi.com/) and [New York Times](https://developer.nytimes.com/) APIs.

### Class Objectives

By the end of this lesson, the students will be able to:

* Make `get` requests with the Request library.

* Convert JSON into a Python dictionary.

* Read and apply API documentation.

* Sign up for and use an API key.
